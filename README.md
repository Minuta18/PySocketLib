# PySocketLib

PySocketLib is a free and open-source cross-platform library to simplify working with sockets. Windows, MacOS and Linux are supported. Installation:

```bash
pip install PySocketLib
```

## Features

- TCP, UDP
- IPv4 and IPv6 support
